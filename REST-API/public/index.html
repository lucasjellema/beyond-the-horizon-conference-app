Search Tags

Post Request to https://data-api-lucasjellema.apaas.em2.oraclecloud.com/tags Send JSON message in body: 
{ "tagCategories" : ["content","duration"]
 , "filterTags" : ["SOA","OSB"]
 , "searchTerm": ""
}

- Retrieve Tags for Sessions that have all of the specified filterTags associated with them
- Retrieve Tags for one of the specified categories (or all categories if no value is indicates)
- Retrieve tags satisfying the search term with part of their display title
For all tags, show the number of occurrences (sessions) for all tags and sessions that satisfy the search conditions
